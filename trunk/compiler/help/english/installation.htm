Perimeter maps are destributed in files with the extension .shrub. The Garden at the site of clan RUL is one place where one can find some Shrubs to download.

Preview And Installation

To preview a Shrub, select Unpack Shrub from the File menu. This process may take a while, and when it is done, you will see the map in the Preview panel. If you like what you see, you can then install the map by selecting Install Map from the File menu.

Installation does not require a connection to the internet. All you need is to have Perimeter installed.
