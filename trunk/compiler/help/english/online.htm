Online Play

Because Perimeter has not been designed for installation of new maps, one more step has to be done in order to enable it to play new maps online.

Perimeter Map Compiler distribution comes with a file named Perimeter.exe. All you have to do is to go into the game folder, rename the file named Perimeter.exe there to RealPerimeter.exe, and copy the Perimeter.exe from the Perimeter Map Compiler distribution there. That's all.

You do not have to know this, but if you are curious about what this accomplishes, it is this. The new Perimeter.exe checks whether an internet connection is present. If it is, it downloads the list of existing maps, and synchronizes Perimeter with it. It then goes on to start RealPerimeter.exe normally.

If you decide to install a patch for Perimeter, you have to reverse the steps, as the patch will likely not know of the switch. Remove the file Perimeter.exe from the folder, and rename RealPerimeter.exe back to Perimeter.exe before installing the patch.
