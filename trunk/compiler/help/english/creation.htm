Map Creation Overview

The philosophy of the Perimeter Map Compiler dictates that it should not attempt to do things it can never do better than software created specifically for doing them. These things include texture and heightmap editing.

When you create a new project, Perimeter Map Compiler will create two images for you - texture.bmp and heightmap.bmp. The texture bitmap defines the colour of the map at each point. The heightmap bitmap is a grayscale image defining the height of the map at each point. Here, lighter means higher, darker means lower, and where the heightmap is black, land does not exist.

You can edit these images with any image editor you like. You can also use specialized texture and heightmap creation software, if you wish. If you do not have any of these, there are some free tools available:

  • The GIMP is a capable open-source image editor.
  • Skencil is an open-source vector drawing application. You can use it for design.
  • Terragen is a terrain generator. With some plugins, the links to which you can find on its site, terrains can be exported as bitmaps.
