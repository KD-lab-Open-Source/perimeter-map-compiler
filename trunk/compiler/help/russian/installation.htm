Кусты

Кустами мы зовём наших упакованных неприкаянных. Пакуют их для того чтобы чего не учудили.
Обурец

Карты для Периметра распространяются как файлы с расширением .shrub. Кусты можно найти в Огороде сайта клана РУЛ.

Предпросмотр и Установка

Для того, чтобы посмотреть на содержимое Куста, нужно выбрать команду Unpack Shrub из меню File. Это займёт некоторое время, по прошествии которого карта появится в панели Preview. Если карта хороша, её нужно установить, выбрав команду Install Map из меню File.

Для установки не требуется сетевое соединение. Всё что нужно, это установленный Периметр.
