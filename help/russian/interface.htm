Оболочка

Оболочка Компилятора Карт Периметра состоит из главного окна и трёх панелей, управляемых кнопками. Панель Preview показывает трёхмерную модель карты. В настройках программы можно включать и выключать освещение, так как оно значительно влияет на скорость загрузки карты. Панель Details даёт возможность настроить карту. Панель Statistics показывает среднюю высоту и цвет карты. Эти данные могут пригодиться для выбора высоты зерослоя и цвета тумана.

Следует учитывать что панелям нужно время, чтобы генерировать информацию, которую они показывают. Панель Preview наиболее "дорогая", а панель Statistics на втором месте. Отключение панели может ускорить загрузку.

Компилятор Карт Периметра использует особую архитектуру для ускорения времени отклика на ваши действия. Когда вы даёте ему команду, вроде загрузки Куста, он останавливается на мнгновение, чтобы загрузить информацию, необходимую вам для принятия дальнейших действий, и затем делает оставшуюся сложную работу в фоновом режиме. В это время вы можете спокойно задавать ему новые задания. Когда приложение занято, иконка с правой стороны главного окна меняется от зелёного к оранжевому.

Если Компилятор Карт Периметра держать открытым во время редактирования файлов карты, он будет следить за этими файлами, и обновлять панели, когда любая из них меняется. Это позволяет вам видеть результат во время работы. Следует заметить, что для того чтобы Компилятор мог увидеть изменение, файл необходимо сохранить.

Меню

  • File
    • New Project: Новый Проект - создать новую карту
    • Open Project: Открыть Проект - открыть начатый проект создания карты
    • Pack Shrub: Упаковать Куст - создать и зарегистрировать законченную карту
    • Unpack Shrub: Распаковать Куст - открыть карту, с тем, чтобы, возможно установить её
    • Install Map: Установить Карту - установить проект или куст
    • Manage Maps: Просмотреть Карты - список установленных карт
    • Project Settings: Настройки Проекта - здесь можно включать и выключать дополнительные возможности
    • Exit: Выход
  • Tools
    • Preferences: Настройки - влияют на качество модели предосмотра, и на качество создаваемых кустов
      • Texture colour tradeoff: Компропис цвета текстуры - Компилятор может создавать текстуры либо быстро, либо качественно.
      • Mesh tolerance: Погрешность сетки - Модель предпросмотра может создаваться с различной точностью. Чем меньше погрешность, тем более точна модель, тем более мощная конфигурация нужна для её просмотра.
      • Zero-level colour: Цвет нулевого уровня - Нулевой уровень показывается в окне предпросмотра в виде полупрозрачного квадрата наложенного на карту. Это его цвет.
      • Zero-level opacity: Прозрачность нулевого уровня - См. выше.
      • Shadows and lighting: Тени и освещение - Просчёт теней и освещения значительно влияет на скорость загрузки. Иногда имеет смысл отключать их.
      • Display as map texture: Показывать в качестве текстуры карты - Выбор предпросмотра различных данных - текстуры, жёсткости или зерослоя.
    • Save Thumbnail: Сохранить Фотографию - сохранить вид карты сверху

    • Help (English): Руководство (Английское) - это читают другие
