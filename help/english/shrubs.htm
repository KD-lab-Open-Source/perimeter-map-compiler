Project and Shrubs

To start a new map you need to create a new project. Once you finish the project, you pack it into a Shrub - the file ready for distribution.

When working on a project, you can install it to see how it looks within the game. Only one project is installed at any one time, and generally, you cannot play it with other players online. An installed project has its name prefixed with [U] in the in-game map list, to distinguish it from normal maps.

When you are done, and are sure that your map is perfect, you can pack it into a Shrub. This requires an internet connection, as the Shrub has to be registered online. This is required for synchronization between players during online play. Only a few bytes are sent during this registration. Absolutely no personal information is transmitted. You can register a map several times, but only as long as you do not change it. If you change it, you have to change its name before registering it again. This is to prevent existence of different maps of the same name.

The shrub you create contains all information about the map. You can install it, give it to a friend, or put it up for download. Clan RUL accepts Shrubs into its Garden. Not all Shrubs are accepted, however, as a certain level of quality has to be met.
